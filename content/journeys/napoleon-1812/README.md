# napoleon-1812 (free flagship)

Content for Napoleon's 1812 Russian campaign — the free flagship journey and the
quality bar for everything else (HANDOFF.md §12).

To be authored in **Phase 1**:

- `journey.json` — manifest + daily-resolution schedule (~170 days), Minard-sourced
  strength curve, temperature series, 60–100 moments, episodes (Niemen, Smolensk,
  Borodino, Moscow, Maloyaroslavets, Krasny, Berezina, Vilna).
- `routes/grande-armee.geojson` — dense route following period roads
  (Kaunas–Vilnius–Smolensk–Borodino–Moscow + retreat variant).
- `routes/russian-army.geojson`
- `labels.geojson` — period place names (Kowno, not Kaunas).
- `scenes/` — ~15–25 curated stills + `art-direction.md`.
- `ATTRIBUTION.md` — sources and licenses for every asset.

The proof-of-concept `events/napoleon_1812.json` (in the archived POC directory) is
**seed reference only** — the structure ports over, the fidelity must be upgraded.
